# NIST CAVP test vectors

Files in this directory are test vectors published by NIST's Cryptographic Algorithm Validation Program (CAVP). NIST publications are in the United States public domain.

**Source:** https://csrc.nist.gov/projects/cryptographic-algorithm-validation-program/cavp-testing-block-cipher-modes ("GCM Test Vectors" / `gcmtestvectors.zip`)

**Files:**
- `gcmEncryptExtIV256.rsp` — AES-256 GCM encryption test vectors with externally-supplied IVs.
- `gcmDecrypt256.rsp` — AES-256 GCM decryption test vectors, including FAIL cases.

**Downloaded:** 2026-04-16 (record the actual date when pulling).

These vectors are not modified from the NIST originals.
